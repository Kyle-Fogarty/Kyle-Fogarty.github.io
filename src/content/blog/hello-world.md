I've rebuilt my personal site from scratch. The old one served me well, but it was time for something cleaner — a single page that puts research front and centre, with a design language inspired by soft watercolor pastels and generous whitespace.

## What's here

- **Publications** — kept up to date from a single data file.
- **News** — short updates on papers, talks, and projects.
- **This blog** — occasional notes and writeups on 3D graphics, vision, and machine learning.

The site is built with [Astro](https://astro.build), ships zero JavaScript beyond a theme toggle, and is deployed on GitHub Pages.
